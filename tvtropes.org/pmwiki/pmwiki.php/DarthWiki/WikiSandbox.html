Doing A Quick Test To See If An All Red Entry Works Now All red entry is an article made out of red links. Of course, this is a lot easier to do than this article's evil-er twin, all blue entry, because you can just internal link every word, Or Wiki Word Every Thing Here, or pothole some nonsense that you made up. However, if someone makes any of these into an article, even if it's not in the same namespace, we're F'd. As an example, while this article used to be a Self-Demonstrating Article, it's not anymore. You used to be able to avoid this by linking to uneditable articles that show up as a red link, like This Troper. But now they turn blue anyways sometimes. Thanks A Lot TV Tropes. Please note: this will hurt your eyes, unlike all blue entry.
