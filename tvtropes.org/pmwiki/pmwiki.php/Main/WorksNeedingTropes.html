TV Tropes is mostly about two things: Listing up the examples of works wherein a trope is used, and listing up the tropes used in works. Put these two together, and they form a nice network that encourages long Wiki Walks.

This page is about the latter half: Works which lack a good list of tropes. If you have found such a page, add it to this list. If you have fixed a page from this list, remove it. Same thing applies if you find that someone else added tropes to a work, of course.
