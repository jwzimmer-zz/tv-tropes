Action Game

Go To

Edit Locked

  • Main
  • More Less
  • Create New

Essentially, an Action Game is a video game with a primary focus on physical challenge above all else, and is usually characterized by fast-paced, intense gameplay that demands speed, mental dexterity and hand-eye coordination in lieu of exploration or puzzle-solving. Combat and/or obstacle crossing are heavily featured, if not the point of the game.

Technically speaking, many genres, including platformers, Action-Adventure games, first-person shooters and Beat 'em Up games can be considered subsets of the action genre, although they are generally seen as distinct from straight-up action games.
