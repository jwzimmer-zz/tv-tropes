Ghost Fiction

Go To

Edit Locked

  • Main
  • Laconic
  • Create New

Advertisement:

This index is for works that focus on ghosts. Just having a ghost does not qualify. Instead, the ghost should be one of the main characters, the main threat, or otherwise be central to the plot (as in Hamlet, where it's the ghost that kickstarts the story). Even if their ghosts are different, if they're central to the plot, add them here.

Also see the related indexes Mummy Media, Vampire Fiction, Werewolf Works, Witch Works, and Zombie Stories.
