Unisex Tropes

Go To

Edit Locked

  • Main
  • Laconic
  • Create New

These are tropes that have a gender-specific name but can involve either one.

Although some will have more examples of one gender than the other, it's not to the degree of the tropes we list as "Always Male/Female".

It also means it's a mis-play to list "rare male/female example" on any entry in the trope pages below.

Again, the trope title must have something that makes it look like it's about just one gender.

Advertisement:

Note that using Pothole markup to replace a trope's name with a more fitting one in a trope list is considered a Sink Hole and not allowed. Many of these tropes have opposite-gender or gender-neutral versions of their name as redirects, so feel free to use them instead (or make one yourself if it doesn't exist).

Compare Non-Indicative Name, Never Trust a Title.

Contrast Always Female, Always Male (both of which actually are about gender-specific tropes), Double Standard.

Not to be confused with Gender-Blending Tropes, Gender-Inverted Trope (as both of these pages deal with toying around with gender tropes, not just trope names).
