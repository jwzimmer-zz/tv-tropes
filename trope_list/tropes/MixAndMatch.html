Mix and Match

Go To

Edit Locked

  • Main
  • NamesToRunAway…
  • Create New

It ain't calculus.
Advertisement:

Take one existing show genre, add another, and presto! you've got a brand-new genre.

The most famous example is Star Trek: The Original Series, pitched as a Wagon Train to the Stars (The Western + Space Opera).

See also X Meets Y (for when it's works, not genres/tropes) and Recycled In SPACE (for when it's a work plus a genre or tropes). Compare and contrast Genre-Busting.

Examples

    open/close all folders 

    Anime and Manga 
  • High School Drama + Lighthearted Comedy + Harem Genre + Shounen + Every Magic Trope Imaginable + Every Science Fiction Trope Imaginable X Serial Escalation = Mahou Sensei Negima!. You may think that's an exaggeration. You would be wrong.
  • High School drama + Humongous Mecha + Moe + Star Crossed Lovers + Transformation Trauma + Dysfunction Junction + Tear Jerker + Downer Ending + Kill 'Em All + Nightmare Fuel Fridge Logic = Saikano.
  • Magical Girl Warriors + Devices from Humongous Mecha + Shounen Combat + Surprisingly Dark Plot + The Military + In Space = Magical Girl Lyrical Nanoha.
  • Magical Girls + Toku + Softcore Cinemax Skinflick = Ultimate Girls.
    • Magical Girl can be divided further; for example, Magical Girl Warrior + Magic Idol = Mermaid Melody Pichi Pichi Pitch.
  • Girls' Love + Humongous Mecha = Kannazuki no Miko.
  • Mind Screw + Humongous Mecha + Viewers Are Geniuses + Eldritch Abomination + Rule of Symbolism + Dysfunction Junction + Deconstruction + Nightmare Fuel = Neon Genesis Evangelion.
  • High School Sitcom + Boy Meets Girl + Sci-Fi + Slice of Life + Locked Room Mystery = The Melancholy Of Haruhi Suzumiya (anime). Well, yeah.
    • Widget Series + Sci-Fi + Mind Screw + Alternate Universe = The Vanishment Of Haruhi Suzumiya.
  • Magical Girl Warrior + Gender Bender = Mahou Shounen Majorian.
  • The Western + Hammer Horror + After the End = Vampire Hunter D.
  • Gentleman Thief + Coming-of-Age Story + Steampunk = The Daughter of Twenty Faces.
  • Magical Girls + Nightmare Fuel + Faustian Rebellion= Puella Magi Madoka Magica.
  • Magical Girls + RPGs + Humongous Mecha = Magic Knight Rayearth.
  • Lolis + Mind Rape + unreliable narrators + The Power of Friendship + What Could Possibly Go Wrong? + Gorn + Moment of Awesome + Awesome Music = Higurashi: When They Cry.
    • Trolls + Fair-Play Whodunnit + Clueless Mystery + Closed Circle + The Power of Love + Awesome Music + Gorn + Serial Escalation+ Maybe Magic, Maybe Mundane + Language of Truth + Clap Your Hands If You Believe = Umineko: When They Cry.
  • Magic Idol Singer + Wholesome Crossdresser = Brocken Blood.
  • High school drama + Feudal Future + Knight in Shining Armor + Humongous Mecha + Serial Escalation (Magnificent Bastard + Wham Episode)³ = Code Geass.
  • Youkai + New Transfer Student + Beast Man + Fanservice + Marshmallow Hell + More Fanservice = Kanokon.
  • Children Raise You + Coming-of-Age Story + Parents (and Children) As People + Parent with New Paramour + Boys' Love = Sakende Yaruze!.
  • Tsubasa -RESERVoir CHRoNiCLE- (Shōjo (Demographic) - Mind Screw) + Kingdom Hearts (Disney Princess - Metaplot) = Kilala Princess.
  • Alice in Wonderland + Mind Screw + Cast Full of Pretty Boys + Dysfunction Junction = PandoraHearts.
  • (Magical Girl + Ballet + Fairy Tales) x (Mad Artist + Rewriting Reality) = Princess Tutu.
  • Ultraman + (Deconstruction + Anyone Can Die + Crapsack World + Alternate History + Redshirt Army) = Attack on Titan.
  • Magical Girl + Sentai = Sailor Moon.
  • Gundam x (Reconstruction + Reincarnate in Another World + High Fantasy + Science Fantasy) = Knight's & Magic

    Comic Books 
  • Medical Drama + The DCU = Samaritan X.

    Film 
  • Seltzer and Friedberg + Black Comedy + Medium Awareness+ Deconstruction x Horror Tropes + Joss Whedon + Subtlety = The Cabin in the Woods.
  • The Silence of the Lambs + Aliens + Stagecoach = Pitch Black.
  • Ditto the Sequel, even if it made a Genre Shift: Star Wars + Alien³ + Conan the Barbarian + Matrix = The Chronicles of Riddick.
  • Body Horrors + Mind Screw = The Soul Taker.
  • Gundam + Cthulhu Mythos + that bloke that did Hellboy = Pacific Rim.
  • Blaxploitation + Our Vampires Are Different = Blacula.
  • Serial Killer + Buddy Cop Show = Se7en.
  • Sexploitation/Erotica + Crime Mystery Film Noir + Slasher Movie = Giallo.
  • Horny Vikings + Instant Awesome: Just Add Dragons! = How to Train Your Dragon.
  • (Fallout 3 - I Love Nuclear Power) + (The Drifter x Badass Preacher x Disability Superpower) + (The Bible x Can't Stop the Signal) = The Book of Eli.
  • A Street Car Named Desire + Gender Flip + So Bad, It's Good = The Room.
  • Dora the Explorer - Edutainment Show + Darker and Edgier + Tomb Raider + Freakin' Machete! = Dora and the Lost City of Gold.
  • Superman - Superhero Tropes + Slasher Movie + Kids Are Cruel = BrightBurn.

    Literature 
  • Government Procedural + Boarding School of Horrors = Gormenghast.
  • Great Detective + Affectionate Parody + Romance Arc = Lord Peter Wimsey.
  • Our Souls Are Different + Waxing Philosophic + Rule of Cool + Film Noir with a Bifauxnen Jerk with a Heart of Gold protagonist = Kara no Kyoukai.
  • Film Noir + Functional Magic + (Badass Longcoat + Genre Savvy + Occult Detective + First-Person Smartass + The Nicknamer) + lots of Shout Outs = The Dresden Files.
  • (Pokémon + Lost Roman Legion) + Five Races + Succession Crisis = Codex Alera. Super extra bonus points for the ones in brackets not only being this trope deliberately invoked, but the very reason for writing the book.
  • Punk Punk settings usually start as Cyberpunk mixed with another genre. For example, Cyberpunk + Medieval European Fantasy = Dungeon Punk.
  • Low Fantasy + (High Fantasy - Hobbits) + Succession Crisis + Character Depth + (Noughties Drama Series - Turn of the Millennium) + 27 * Magnificent Bastard = A Song of Ice and Fire.
  • Redwall + Zootopia + Usagi Yojimbo + Princess Mononoke + Alternate Universe + Beast Fable + What Do You Mean, It's Not for Kids? = Tasakeru.
  • (Xenofiction - Cats Are Mean + Anyone Can Die + Family-Unfriendly Violence) x (Proud Warrior Race Guy + [The Clan x 4]) x (Long-Runners + Loads and Loads of Characters) = Warrior Cats.
  • Weird West + High Fantasy + Horror + Science Fiction = The Dark Tower.
  • Dungeon Punk + Comedy = Discworld
    • ... + Police Procedural = City Watch subseries.

    Live-Action TV 
  • Police Procedural + Prime Time Soap = Crime Time Soap (Hill Street Blues).
  • Sitcom + Wagon Train to the Stars = Red Dwarf.
  • Spy Drama + The Western = The Wild Wild West.
  • Medical Drama + The Western = Dr. Quinn, Medicine Woman.
  • Medical Drama + Amateur Sleuth = Diagnosis: Murder.
  • Medical Drama + Space Opera = Mercy Point.
  • Medical Drama + Work Com = Scrubs.
  • Medical Drama + Work Com + During the War = M*A*S*H.
  • Medical Drama + Rescue = Emergency!.
  • The Musical + Dramedy = Glee.
  • Police Procedural + Work Com = Barney Miller.
  • Police Procedural + Law Procedural =
    • Most famously, Law & Order.
    • ... which was preceded by the short-lived 1971 Jack Webb production The D.A., which used the same basic format.
    • ... which itself was preceded by Arrest And Trial, a 90-minute 1963 series.
  • Police Procedural + Rescue = Third Watch.
  • Police Procedural + Odd Couple * Gender Flip + Government Conspiracy * Science Fiction = The X-Files.
  • (The Western + Sci-Fi = Space Western) + Government Conspiracy = Firefly.
  • Travelogue Show + Reality Show = The Amazing Race.
  • Super Sentai + (Science Ninja Team Gatchaman + Darker and Edgier) + Dorama = Choujin Sentai Jetman.
    • Super Sentai + Everything Is Better With Dinosaurs = Kyoryu Sentai Zyuranger.
    • Super Sentai + Cop Show = Tokusou Sentai Dekaranger.
    • Super Sentai + Wuxia = Juken Sentai Gekiranger.
    • Super Sentai + Jidaigeki = Samurai Sentai Shinkenger.
    • Super Sentai + Milestone Celebration + (Space Pirates * Power Copying) = Kaizoku Sentai Gokaiger.
  • Film Noir + Post-Cyberpunk + Kamen Rider = Kamen Rider Double.
    • Hammer Horror + Kamen Rider = Kamen Rider Kiva.
    • Milestone Celebration + Alternate Universe + (Kamen Rider * Dimensional Traveler * Power Copying) = Kamen Rider Decade
    • High School + Kamen Rider = Kamen Rider Fourze.
    • Coming-of-Age Story + Cosmic Horror Story + Mood Whiplash + Kamen Rider = Kamen Rider Gaim.
    • Robot War + Police Procedural + Kamen Rider = Kamen Rider Drive.
    • Medical Drama + Video Game Tropes + Kamen Rider = Kamen Rider Ex-Aid.
    • Milestone Celebration + (Time Travel * Bad Future) + (Kamen Rider Decade * Tsukasa Returning) + (Kamen Rider * Power Copying) = Kamen Rider Zi-O
  • Kamen Rider + Humongous Mecha + Romance of the Three Kingdoms = Legend Heroes.
  • Super Sentai + Americanization + Media Adaptation Tropes = Power Rangers.
    • Kyoryu Sentai Zyuranger + (Gosei Sentai Dairanger - The Core Five Costumes) + Ninja Sentai Kakuranger + Wake Up, Go to School, Save the World = Mighty Morphin' Power Rangers.
    • Denji Sentai Megaranger - Video Game Tropes + Space Opera + Rescue Arc + Grand Finale = Power Rangers in Space.
    • Seijuu Sentai Gingaman + Wagon Train to the Stars + Darker and Edgier = Power Rangers Lost Galaxy
    • Mirai Sentai Time Ranger + Adaptation Distillation = Power Rangers Time Force.
    • Bakuryuu Sentai Abaranger + Wake Up, Go to School, Save the World + TOMMY = Power Rangers Dino Thunder.
    • Engine Sentai Go-onger - Lighter and Softer + Darker and Edgier + After the End = Power Rangers RPM.
  • "Die Hard" on an X + Race Against the Clock + Real Time = 24.
  • (Heroic Fantasy + Recycled In Space - Two-Fisted Tales) + Walking the Earth + (Shell-Shocked Veteran / [Cool Old Guy x Really 700 Years Old]) + (Timey-Wimey Ball x ∞) = Doctor Who (at least, from 2005 on).
  • Wake Up, Go to School, Save the World + Joss Whedon + Dramedy + Angst = Buffy the Vampire Slayer.
    • Vampire Detective Series + Gothic Punk + The Atoner = Angel.
  • Circus Drama + The Western = Frontier Circus.

    Music 
  • Gangsta Rap + Horror Tropes = Horrorcore Rap.

    Tabletop Games 
  • Warhammer + SPACE - Cooperation + Oversized Weaponry - Cooperation + GRIMDARK - Cooperation + Chainswords - Cooperation: Warhammer 40,000.
  • Cyberpunk + High Fantasy + Strange Combinations + Mercenaries: Shadowrun.
  • Old World of Darkness - Modern day setting + Eldritch Abominations/Titans + Fantasy Kitchen Sink + Anime= Exalted.

    Video Games 
  • Space Marines + Super Soldiers + Alien Invasion + The Virus + Zombie Apocalypse + Hopeless War + War Is Hell + Military Science Fiction + Space Opera = Halo.
  • First-Person Shooter + Le Parkour + Action Girl = Mirror's Edge.
  • Cute Monster Girls + Bullet Hell + Awesome Music + Fantasy Kitchen Sink = Touhou.
    • Take part of the above + Serial Escalation = Mushihime-sama.
  • Metroidvania + Nintendo Hard + Guide Dang It! + Goddamned Bats + Demonic Spiders = La-Mulana.
  • Retro-futuristic Science Fiction + Underwater Habitat + Hidden Elf Village + Survival Horror + Art Deco + Anarcho-Capitalistic Dystopia + Genetic Engineering Is the New Nuke + What Measure Is a Non-Human? + System Shock = BioShock.
  • Platform Hell + Nintendo Hard + Ludicrous Gibs + Everything Trying to Kill You + One Hitpoint Wonder = I Wanna Be the Guy.
  • The Legend of Zelda + Bullet Hell = Engage To Jabberwock.
  • The Legend of Zelda + Character Customization + Metroidvania + Crapsack World + Nintendo Hard = Dark Souls.
  • Dark Souls + Gothic Horror + Cosmic Horror Story = Bloodborne
  • Kaiju + Professional Wrestling = King of the Monsters.
  • Raygun Gothic + Zeerust + Eagleland (both flavors) + Alternate History + After the End = Fallout.
  • Roguelike + The Legend of Zelda I + Nausea Fuel + Nightmare Fuel + Thick-Line Animation = The Binding of Isaac.
  • Humongous Mecha + Lovecraft Lite = Demonbane.
  • 4X + Real-Time Strategy = Rise of Nations.
  • First-Person Shooter + Puzzle Game = Portal.
  • Final Fantasy XIII + Chrono Trigger = Final Fantasy XIII-2.
  • Pinball Quest does this with all of its games:
    • The main quest is Pinball + RPG.
    • The "Pop! Pop!" table is Pinball + Billiards + Bowling.
    • "Viva! Golf" is Pinball + Golf.
    • The "Circus" table is Pinball + Slot Machines + Space Invaders (for the lion tamer Mini-Game).
  • Platform Game + Super Speed + Animal Superhero + Mecha Mooks + Science Fantasy = Sonic the Hedgehog
  • Fighting Game + Role-Playing Game + High Fantasy + Partially Civilized Animal + Lauren Faust = Them's Fightin' Herds.
  • Rhythm Game + Roguelike = Crypt Of The Necrodancer
  • Third-Person Shooter + Art Attacker + The '90s = Splatoon
    • All of the above + Cyberpunk + The '80s + Nintendo Hard = Splatoon 2: Octo Expansion

    Web Comics 
  • Supernatural Soap Opera + Romantic Comedy + Different for Girls = Misfile.

    Web Original 
  • (Deep-Immersion Gaming x 11) + Les Yay + The Plan+ Corrupt Corporate Executive + Moral Event Horizon4 + Sliding Scale of Idealism vs. Cynicism / Loads and Loads of Characters - Rule of Cool + Shown Their Work = KIKEN.
  • Archie Comics + Superhero School + Crazy Awesome + Transgender = Whateley Universe.
  • Any random movie + Winnie-the-Pooh + Any other nostalgic character + Railroading + Dull Surprise + Captain Obvious + Hey, That's My Line! + Forgot About His Powers = Poohs Adventures.
  • The Runaway Guys - 1 = Game Grumps.
  • Let's Play + MST3K = Retsupurae.
  • ([Teen Drama x (Homage + The '90s)] + Elaborate University High) + ([Video Games x Sports Stories] x Serious Business) = Video Game High School.
